# spider-lib

**spider-lib** is an open-source Rust ecosystem for modern web scraping, designed for high performance, modularity, and ease of integration. It follows design patterns similar to Scrapy (Python), with full support for async Rust and concurrency.

## Key Features
- **Asynchronous**: Built with async Rust for fast and scalable scraping.
- **Modular**: Various supporting libraries, such as `spider-core`, `spider-macro`, `spider-middlewares`, etc.
- **Familiar to Scrapy Users**: Adopts a structure that is easy to use for those familiar with Scrapy.
